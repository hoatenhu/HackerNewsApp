# HackerNewsApp

## Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/HackerNewsApp.git
    cd HackerNewsApp
    ```

2. Install dependencies:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn
    ```

4. Run the app:
    ```bash
    npm run android # for Android
    npm run ios # for iOS
    ```
    or
    ```bash
    yarn android # for Android
    yarn ios # for iOS
    ```

## Running Tests

To run tests:
```bash
npm test
```
or
```bash
yarn test
```
